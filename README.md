# 🖋 [景诚] (LJC Blog)

> 当时只道是寻常 —— 记录技术、生活与思考的个人博客。

这是一个基于 [Jekyll](https://jekyllrb.com/) 构建的静态博客项目。原主题基于经典的 [Hux Blog](https://github.com/huxpro/huxpro.github.io) 进行了深度分叉与二次开发，致力于打造**极致的沉浸式阅读体验**。

## ✨ 核心特色 (Features)

本项目在原版的基础上进行了大量前端重构，完美复刻了 Typora 的经典写作风格：

* **🎨 Typora 视觉风格**：
    * 全局采用 **霞鹜文楷 (LXGW WenKai)** 字体，兼具现代感与书卷气。
    * 精心设计的 **H2 标题**：独创的深蓝色块背景与斜切尾部设计，自带极强的视觉引导。
    * **色彩呼应**：链接、行内代码块采用橙红色调 (`rgb(239, 112, 96)`)，与主标题的深蓝色形成优雅的撞色对比。
    * **引用块美化**：带有圆角和主题色左边框的半透明引用区块。
* **📑 树状结构侧边大纲**：
    * 针对长篇文章优化的侧边目录 (TOC)。
    * 独家绘制的**父子层级引导线**，配合动态悬停与高亮效果（蓝色胶囊样式），文章结构一目了然。
    * 采用高性能的 JS 动画进行平滑滚动跳转，告别生硬的页面闪烁。
* **🔍 沉浸式毛玻璃搜索**：
    * 全屏搜索覆盖层，加入磨砂玻璃模糊效果，搜索过程专注且优雅。
* **📱 响应式布局**：
    * 无论是在 4K 显示器还是移动端设备上，正文宽度、行高（`1.6`）和字号（`18px`）均经过严格的“黄金比例”调优，久读不累。
